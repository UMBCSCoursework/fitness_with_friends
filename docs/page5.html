Implementation Instructions
  1. The link will send you to a FlutLab - a Flutter online IDE, that allows to create and build cross-platform projects.

  2. It may ask to create an account, this is free.
    • It seems like this IDE is still under development, and a free account with limited options is the only way to operate.
      A premium account is on hold, and for educational account it is impossible to verify an email,
      because a verification link is never sent to edu mail.

  3. A Web Emulator should start automatically.
    If not, there is a phone icon in the top left corner inbetween "hot reload" and "save" icons.

  4. It is highly recomended to test this on a PC, Mac, or Linux OS.

    • It is possible to run it on a phone.
      However, the scaling of a web emulator is not right and it produces distortions in some cases.

    • Our home screen contains a carousel, i.e. a swipable horizontal list.
      The web emulator reacts only on touchpad for swipe options,
      i.e. a mouse should be over the area of this list (big icons "friend" or "stranger")
      and then a swipe on a touchpad to the left will reveal a third option "instructor".
